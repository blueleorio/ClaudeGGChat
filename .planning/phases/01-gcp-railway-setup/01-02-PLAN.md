  <how-to-verify>
Work through the GCP Console steps in this exact order (sequence matters — each step depends on the previous):

**Step 1 — GCP project:**
1. Go to https://console.cloud.google.com
2. Create a new project or select an existing one
3. Note the **Project Number** (numeric) from Home > Project info panel — this is NOT the Project ID. Example: `123456789012`

**Step 2 — Enable Google Chat API:**
4. APIs & Services > Library > search "Google Chat API" > Enable

**Step 3 — Create service account:**
5. IAM & Admin > Service Accounts > Create Service Account
6. Name: "claudeggchat-bot" (or similar), click Create and Continue
7. No roles needed at this stage — click Done

**Step 4 — Download JSON key:**
8. Click the service account you just created > Keys tab > Add Key > Create new key > JSON > Create
9. The JSON file downloads automatically — **do not close this tab until you've completed Step 5**

**Step 5 — Store key in Railway immediately:**
10. In Railway: Service > Variables > Raw Editor
11. Add variable: `GOOGLE_SERVICE_ACCOUNT_KEY` = paste the entire contents of the downloaded JSON file
12. Add variable: `GOOGLE_CLOUD_PROJECT_NUMBER` = the numeric project number from Step 3 (stored as a string)
13. The local JSON key file can now be deleted (it is safely stored in Railway)

**Step 6 — Configure Chat app:**
14. APIs & Services > Google Chat API > Configuration tab
15. Fill in:
    - App name: `ClaudeGGChat`
    - Avatar URL: any publicly accessible image (e.g., `https://developers.google.com/workspace/chat/images/chat-product-icon.png`)
    - Description: `Claude assistant for SEV team`
    - Connection settings: HTTP endpoint URL = your Railway HTTPS URL from Task 2
    - **Authentication Audience: select "Project Number"** (NOT "HTTP endpoint URL" — critical for Phase 2 JWT verification)
16. Slash commands section > Add a slash command:
    - Command ID: `1`
    - Command name: `/claude`
    - Description: `Ask Claude a question`
    - Hint text: `[your question]`
17. Save the configuration

**Step 7 — Set remaining Railway env vars:**
18. In Railway Variables, also add:
    - `ANTHROPIC_API_KEY` = your Anthropic API key (from https://console.anthropic.com)
    - `ALLOWED_SPACE_IDS` = the space ID of your authorized Google Chat test space

    To find a space ID: In Google Chat, open the space, click the space name > Manage webhooks or inspect the URL — the space ID is the `spaces/XXXXXXX` portion.
